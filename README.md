# Introduction to Computer Graphics

This repository contains the source files used in the "Introduction to Computer Graphics" course, which provides a fundamental understanding of computer graphics and its application in game development. The curriculum covers a mathematics primer, 2D and 3D graphics, shading, and hands‑on experience with practical tools such as p5.js and Blender.

**Course Overview**

- Topics: mathematics primer, 2D/3D graphics, shading, practical tools
- Assessment: mandatory assignments, a final project, and a final report
- Schedule: sessions start s.t. (sine tempore) in Klagenfurt local time; see the syllabus for details

**Key Resources**

- Source code and assets are hosted on GitHub: <https://github.com/dermotte/introduction-to-computer-graphics-lecture>
- Course materials, including slides and additional links, are available iat <https://www.itec.aau.at/~mlux/?data=courses/intro-cg-ss26.md>

## How to use the files in this repository

The JavaScript files in this repository are for the p5js framework. You can copy the sources over to <https://editor.p5js.org/> and try them out there.
